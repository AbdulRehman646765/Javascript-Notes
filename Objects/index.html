Objects in JavaScript

There is Only one Type of Non Primitive Data Type that is Called Object.

Objects are Further Divided into Three Categories.

  1. Object Literals

    2. An object in Javascript is an entity having properties and methods. Everything is an object in javascript.

    Syntax Of Objects

              const objectName = {
            Property: PropertyValue,
            Property: PropertyValue
        }

    Object's Example

            Example: Below is examples.
  
        const user = {
          firstName: "Abdul", 
          lastName: "Rehman",
          fatherName: "Muhammad Mehdi",
          age: 23,
          isEducate: true,
      }
        console.log(user);
          // The Result is "{firstName: 'Abdul', lastName: 'Rehman', fatherName: 'Muhammad Mehdi', age: 23, isEducate: true}"

    Another Object's Example

              const myIntro = {
            firstName: "Abdul",
            lastName: "Rehman",

          }
    You can access object properties in two ways:
    1. Dot Notation
    2. Square Bracket Notation

    Dot Notation

            console.log(myIntro.firstName); // The Result is "Abdul"

    Square Bracket Notation

              console.log(myIntro["firstName"]); // The Result is "Abdul"

              let userIntro = {

    Object. seal() creates a “sealed” object, while Object. freeze() creates a “frozen” object. Sealed objects are still mutable, whereas frozen objects are completely immutable. Both methods return the modified object itself.

    Object.seal()

              const userIntro = {
            firstName: "Abdul",
            lastName: "Rehman",
            age: 23,
        }
          Object.seal(userIntro)
          userIntro.lastName = "Qadeer" // This Change Property
          delete userIntro.lastName // This Delete Property
          userIntro.isGraduate = true // This Add New Property
          console.log(userIntro);
          // The Result is "{firstName: 'Abdul', lastName: 'Qadeer', age: 23}"

    Object.freeze()

              const userIntro = {
            firstName: "Abdul",
            lastName: "Rehman",
            age: 23,
        }
          Object.freeze(userIntro)
          userIntro.lastName = "Qadeer" // This Change Property
          delete userIntro.age // This Delete Property
          userIntro.isGraduate = true // This Add New Property
          console.log(userIntro);
          // The Result is "{firstName: 'Abdul', lastName: 'Rehman', age: 23}"
