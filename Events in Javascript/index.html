Events in JavaScript

Add Event this way

Example of Event

        Example: Below is an example.

        function myFuc(){
            console.log("Clicked on Main Title");
        }
        const mainTitle = document.querySelector(".main-title")
        mainTitle.onclick = myFuc

    Output:

        "Clicked on Main Title"

addEventListener as a Named Function

Example of addEventListener as a Named Function

        Example: Below is an example.

        function sayHi(){
            console.log("Clicked on Heading Number Two");
        }
        const h2 = document.querySelector("h2")
        h2.addEventListener('click', sayHi)
        
    Output:

    "Clicked on Heading Number Two"

addEventListener as a Anonymous Function

Example of addEventListener as a Anonymous Function

        Example: Below is an example.

        const card = document.querySelector(".card")
        card.addEventListener("click", function(){
            console.log("Clicked on this Card Number");
        })
        
    Output:

    "Clicked on this Card Number"

addEventListener as a Arrow Function

Example of addEventListener as a Arrow Function

        Example: Below is an example.

        const card2 = document.querySelector(".card")
        card2.addEventListener("click", () => {
            console.log("Clicked on this Card Number");
        })
        
    Output:

    "Clicked on this Card Number"
