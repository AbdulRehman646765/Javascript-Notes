Data Types in JavaScript

There are two types of data types in JavaScript.

  1. Primitive ( Value Types )
  2. Non Primitive ( Reference Types )

Primitive Data Types in 7 Categories.

      NN : Number & Null 
      SS : String & Symbol
      BB : Boolean & BigInt  
      U  : Undefined
  1. Number

    2. Number data type in javascript can be used to hold decimal values as well as values without decimals. 

          Example: Below is an example.

      const x = 786;
      const y = 25.26;
      console.log("Value of x = " + x); // The Result is "Value of x = 786"
      console.log("Value of y = " + y); // The Result is "Value of y = 25.26"

  2. Null

    3. This data type can hold only one possible value that is null.

          Example: Below is an example.

      const x = null;
      console.log("Value of x = " + x); // The Result is "Value of x = null"

  3. String

    4. The string data type in JavaScript represents a sequence of characters that are surrounded by single or double quotes. 

          Example: Below is an example.

      const string1 = 'Hello Friends';
      const string2 = "Welcome to My New House";
      console.log("Value of string1 = " + string1); // The Result is "Value of string1 = Hello Friends"
      console.log("Value of string2 = " + string2); // The Result is "Value of string2 = Welcome to My New House"

  4. Symbol

    5. This data type is used to create objects which will always be unique. these objects can be created using Symbol constructor. 

          Example: Below is an example.

      const sym = Symbol("Hello") // Symbol Define karny ky lye "S" ko Capital rakhna ho ga.
      console.log(typeof (sym)); // The Result is "symbol"
      console.log(sym); // The Result is "Symbol(Hello)"

  5. Boolean

    6. The boolean data type can accept only two values i.e. true and false. 

          Example: Below is an example.

      const username = "Abdul Rehman";
      const userAge = 23;
      const isGraduate = true;
      console.log(isGraduate); // The Result is "true"
      console.log(typeof isGraduate); // The Result is "boolean"

  6. BigInt

    7. This data type can represent numbers greater than 253-1 which helps to perform operations on large numbers.
    The number is specified by writing ‘n’ at the end of the value. 

          Example: Below is an example.

      Without BigInt

      const x = 100000000000
      const y = 100000000000
      console.log(x*y); // The Result is "1e+22"
      
      With BigInt
      
      const x = 100000000000n
      const y = 100000000000n
      console.log(x*y); // The Result is "10000000000000000000000n"

      Note! BigInt ki help sy hum bari sy bari value ko b calculate kar sakty hain.
      Note! But BigInt ki help sy calculate usi zoorat main ho ga ja hum value ky end main "n" lagayn gy.

  7. Undefined

    8. The meaning of undefined is ‘value is not assigned’.

          Example: Below is an example.

      let x  // value is not assigned 
      console.log("Value of x = " + x); // The Result is "Value of x = undefined"
