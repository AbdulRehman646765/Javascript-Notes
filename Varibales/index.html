Variables in javaScript

JavaScript Variables

Variables are Containers for Storing Data

JavaScript Variables can be declared in 4 ways:

  1. Automatically
  2. Var
  3. Let
  4. Const

Rules for Naming Variables.

A variable name can contain any of the following charaters but must not start with a digit.

  • Capital Letters: A to Z
  • Small Letters: a to z
  • Digits: 0 to 9 (but we can't start variable name with digits)
  • Only two special charatars that are: Dollor "$" and Underscore "_"

When to Use var, let, or const?

  1. Always declare variable.
  2. Always use const if the value should not be changed.
  3. Always use const if the type should not be changed. (Arrays and Objects)
  4. Only use let if you can't use const.
  5. Only use var if you MUST support old browsers.

Syntax Of Variables

    const x = 5;
    const y = 6;
    const z = x+y;
    console.log (z); // Result is 11

Dialog Box Create

3 Way Dialog Box Create

  1. alert ()
  2. confirm ()
  3. prompt ()

How To Create Dialog Box

  
Alert Box Create.

      alert("Please Press OK Button");

  
Confirm Box Create.

      const isConfirm = confirm("Would you like to proceed?");
      console.log(isConfirm);
      
  
Prompt Box Create.

      const userInput = prompt("Please Enter Your Name.");
      console.log(userInput);

Convert String to Number

Some Examples

  let user2Age = "23"

  user2Age = Number(user2Age); // Convert "23" to 23
  user2Age = parseInt(user2Age); // Convert "23" to 23
  user2Age = parseFloat(user2Age); // Convert "23" to 23
  console.log(user2Age, typeof user2Age);
