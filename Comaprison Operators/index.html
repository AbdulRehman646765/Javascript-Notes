Comparison Operators in JavaScript


Comparison Operators

  1. == Equal to (equality)

    2. is condition main 2nu varibales (user1Age & user2Age) ka Equal hona zaroori hai. 

            Example: Below is an example.
            
        let user1Age = 18; 
        let user2Age = 18;
        console.log(user1Age == user2Age); // user1Age is Equal to user2Age.
  2. === Equal Value and Equal Type (strict equality)

    3. is condition main 2nu varibales (user1Age & user2Age) ki value tu equal hai but in ki type equal nahi hai wo is wajha sy ky
    user1Age ki type "Number" hai or user2Age ki type ik "String" hai. 

            Example: Below is an example.

        let user1Age = 18; // This Varibale is a Number
        let user2Age = "18"; // This Varibale is a String
        console.log(user1Age === user2Age);
  3. != Not Equal

    4. is condition main 2nu varibales (user1Age & user2Age) equal tu hain but is operator "!=" ki wajha sy kio ky yeh equal ka oposit
    batata hai isi wajha sy yeh bata raha hai ky yeh equal nahi hai. 

            Example: Below is an example.

        let user1Age = 18;
        let user2Age = 18;
        console.log(user1Age != user2Age);
        // is ka result tu equal ana chahye but is operator "!=" ki wajha sy is ka result not equal hai.
  4. !== Not Equal Value and Equal Type (strict)

    5. yeh condition "!==" opposite hai is condition "===" ky.
