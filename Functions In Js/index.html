Functions in JavaScript

  • Functions

    • JavaScript function is a set of statements that take inputs, do some specific computation, and produce output.

    Syntax Of Function

     
      // Function Definition................
        function functionName (Parameter1, Parameter2, ...) 
        {
          // Function body
        }

      // Function Call................

    Function Example with Default Parameters

          Example: Below is an example.

      function introduceMe(username = "Procoder", skill = "Professional") {
        console.log(`My Name is ${username}`);
        console.log(`I'm a ${skill}`);  
      }
      
      introduceMe("Abdul Rehman", "Web Developer")
      // The Result is "My Name is Abdul Rehman.
                        I'm a Web Developer."
                        
      introduceMe("Bilal Hassan", "Software Engineer")
      // The Result is "My Name is Bilal Hassan.
                        I'm a Software Engineer."

      introduceMe() // Default Parameters Example
      // The Result is "My Name is Procoder.
                        I'm a Professional."

    Arrow Functions

    Before Arrow:

        Example:  Below is an example.

    hello = function() {
      return "Hello World!";
    }

    After Arrow Function:

        Example:  Below is an example.

    hello = () => {
      return "Hello World!";
    }

    Implicit Return:

        Example:  Below is an example.

    add = (a,b) => a + b;

    Explicit Return:

        Example:  Below is an example.

    add = (a,b) => {
      return a + b;
    }
