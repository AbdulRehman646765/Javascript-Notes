ForEach, Map, Filter and Reduce in JavaScript


ForEach Array Method

ForEach loop Syntax:

      array.forEach(function(currentValue, index, arr), thisValue)
      
Parameters:

      function()
                Required.
                A function to run for each array element.
      currentValue
                Required.
                The value of the current element.
      index
                Optional.
                The index of the current element.
      arr
                Optional.
                The array of the current element.
      thisValue
                Optional. Default undefined.
                A value passed to the function as its this value.

Some Examples:

Books Example with anonymous function and arrow function 

      Example: Below is an example.

        const books = ["Urdu", "English", "Islamic Studies", "Physics", "Chemistry"];

        books.forEach((book) => {
          console.log(book);
        })

Animals Example with named function

      Example: Below is an example.

        const animals = ["Horse", "Cat", "Dog", "Tiger", "Lion"];

        function abc(el){
          console.log(el)
        }
        animals.forEach(abc) 

      Note! Named function itna common nai hai jitna anonymous function common hai.

Map Array Method

Map aur ForEach dono same hi hain. Difference sirf itna hai ky Map ki return value hoti hai lekin ForEach ki return value nai hoti.

Map Syntax:

        array.map(function(currentValue, index, arr), thisValue)
        
Parameters:

      function()
                Required.
                A function to run for each array element.
      currentValue
                Required.
                The value of the current element.
      index
                Optional.
                The index of the current element.
      arr
                Optional.
                The array of the current element.
      thisValue
                Optional. Default undefined.
                A value passed to the function as its this value.

Some Examples:

Days Example

      Example: Below is an example.

        const days = ["Monday", "Tuesday", "Wednesday", "Thursday", "Friday", "Saturday", "Sunday",];
        const mapExample = days.map(function(day){
          console.log(day);
        });
        console.log(mapExample);

        
        Return Output:

        (7) [undefined, undefined, undefined, undefined, undefined, undefined, undefined]

      Note! mapExample variable main is ki return value save hai. ager hum is variable ko access karyn gy tu return value show 
             ho gi.

      Note! ager hum kuch b return nai karyn gy tu return main aik array aye ga kis main har value ko undefined print kary ga.

Another Example with Return value

      Example: Below is an example.

        const days = ["Monday", "Tuesday", "Wednesday", "Thursday", "Friday", "Saturday", "Sunday",];
        const capitalMonths = days.map(function(day){

        (7) ['MONDAY', 'TUESDAY', 'WEDNESDAY', 'THURSDAY', 'FRIDAY', 'SATURDAY', 'SUNDAY']

Filter Array Method

Note! Filter method True aur False values ky hesaab sy print karta hai. Jis value ky lye False ho ga usy skip kar dy ga. or jis ky lye True ho ga usy Print kary ga.

Note! Yahan par yeh True aur False nai print kar raha. Balky jis ky lye True aur False hai usy print kar raha hai.

Filter Syntax:

      array.filter(function(currentValue, index, arr), thisValue)

Some Examples:

vegetables Example

      Example: Below is an example.

      const vegetables = ["Garlic", "Ginger", "Onion", "Cucumber", "Tomato"];
      const filteredVegetables = vegetables.filter((vegetable) => {
        console.log(vegetable, ":" ,vegetable.length);
        return vegetable.length < 7;  
      });
      console.log(filteredVegetables); // all values returned but cucumber value skipped. because cucumbers value length is 8.

Another Example

      Note! Below is an example.

      const vegetables = ["Garlic", "Ginger", "Onion", "Cucumber", "Tomato"];
      const filteredVegetables = vegetables.filter((vegetable) => {
        console.log(vegetable);
        return vegetable.toLocaleUpperCase().includes("O") // is main hum jo b word print karyn gy. wo word jis jis value main
                                                              ho ga wo value print ho jye gi.
      });
      console.log(filteredVegetables); // The Result is "(2) ['Onion', 'Tomato']"

Filter index Example

      Note! Below is an example.

      const vegetables = ["Garlic", "Ginger", "Onion", "Cucumber", "Tomato"];

Chaining array method

      Note! Below is an example.

      const students = [
        {
          name: "Ali Hassan",
          age: 29,
        },
        {
          name: "Bilal Hassan",
          age: 27,
        },
        {
          name: "Abdul Rehman",
          age: 18,
        },
        {
          name: "Qasim Hussain",
          age: 17,
        },
        {
          name: "Baber",
          age: 15,
        },
      ];
      const adultStudent = students.filter((student) => {
        return student.age >= 18; // is sy jin ki age 18 yan 18 sy above ho gi wo print ho jyn gy.

      }).map((student) => {
        return student.name; // is sy jo print howy un ky name aa jyn gy. age remove ho jye gi.

      }).filter((student) => {
        return student.includes("A"); // Phir jo jo print howy un main sy sirf un ka name aye ga. jin main "A Capital" ho ga.

      }).map((student) => {
        return student.toUpperCase(); // Phir is sy un ka name upperCase main convert ho jye ga.
      });

      console.log(adultStudent); // The Result is "(2) ['ALI HASSAN', 'ABDUL REHMAN']"

Reduce Array Method

Syntax of Reduce:

      array.reduce(function(total, currentValue, currentIndex, arr), initialValue)    
    
Parameters:

      function()
                Required.
                A function to be run for each element in the array.
      total
                Required.
                The initialValue, or the previously returned value of the function.
      currentValue
                Required.
                The value of the current element.
      currentIndex
                Optional.
                The index of the current element.
      arr
                Optional.
                The array the current element belongs to.
      initialValue
                Optional.
                A value to be passed to the function as the initial value.
