Arguments Keyword in JavaScript

Arguments Keyword

jab hum function ko call karyn gy tu us main pehle parameters ka result aye ga phir arguments ki values array like object main aye gi.

Arguments Keyword ko hum Named Function aur Anonymous Function main pass kar sakty hain, But Arrow Function main use nai kar sakty.

Some Examples:

Add Numbers Example with Named Function

        Example: Below is an example.

        function add(a,b){
            console.log(arguments);
            console.log(a + b);	 
        }
        add(5,6) // The result is "11"

Subtract Numbers Example with Anonymous Function

Ager hum ny jitny b parameters define kye un sy ziaada arguments pass karyn. tu jitny parameters define kye un ka tu result aa jye ga or baqi arguments skip ho jyen gy. 

        Example: Below is an example.

        const subtract = function(a,b){
            console.log(arguments);
            console.log(a - b);	 
        }
        subtract(10,8,7,9) // The result is "2"

        Note! is example main "10 aur 8" ka result tu aya or baqi dono arguments skip ho gye.

Square Numbers Example with Arrow Function

Note! Cannot access it 
	Example: Below is an example.

	const square = (num) => {
		console.log(arguments); // Error: arguments is not defined at square
		console.log(num * num);
	}
	square(12)

Ager arguments ko access karna ho

        Example: Below is an example.

        function subtract() {
            console.log(arguments[0]);
            console.log(arguments[1]);
            console.log(arguments[2]);
            console.log(arguments[3]);
        }
        subtract(7,2,8,5)

        Output:

        7
        2
        8
        5

Arguments Keyword Example with For loop 

        Example: Below is an example.

        function add() {
          let sum = 0
          for (let i = 0; i < arguments.length; i++) {
          sum += arguments[i];
          }
          return sum;
        }
        const result = add(10,20,500,120,100,36);
        console.log(result); // The Result is "786"

Note! Yan par hum ny multiple arguments pass kye or hum in ko use b kar sakty hain. Lekin parameter aik bhi define nai kya.
       Or yeh sab kuch hum Arguments Keyword ki help sy kar rahy hain.
