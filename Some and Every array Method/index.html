Some and Every array Method


Some array Method

Some array Method main ager Truthy value define karyn tu result True aye ga. Ager Falsy Value define karyn tu result False aye ga.

Ager is main multiple values (Truthy and Falsy) define karyn tu isy jesy hi Truthy Value mily gi tu result main True print kar dy ga. or yeh baqi Falsy Values ko nai dekhy ga.


Some array Method Syntax:

        array.some(function(value, index, arr), this)
        
Parameters:

      function()
                Required.
                A function to run for each array element.
      Value
                Required.
                The value of the current element.
      index
                Optional.
                The index of the current element.
      arr
                Optional.
                The array the current element belongs to.
      this
                Optional. Default undefined.
                A value passed to the function to be used as its "this" value.

Some Examples:

Even Numbers Example with Truthy value

      Example: Below is an example.

      const evenNumbers = [0,2,4,6,8];

      const result = evenNumbers.some((num) => {
        return num > 4 // yan par yeh "0,2 aur 4" ky lye False aur "6 aur 8" ky lye True ho ga.
      })
      console.log(result); // The Result is "True"

Even Numbers Example with Falsy value

      Example: Below is an example.
      
      const evenNumbers = [2, 4, 6, 8, 9];

      const result = evenNumbers.some((num) => {
        return 0 // "0" is a Falsy value.
      });
      console.log(result); // The Result is "False"

Every array Method

Opposite of some() array Method

Every array Method main ager Truthy value define karyn tu result True aye ga. Ager Falsy Value define karyn tu result False aye ga.

Ager is main multiple values (Truthy and Falsy) define karyn tu isy jesy hi Falsy Value mily gi tu result main False print kar dy ga. or yeh baqi Truthy Values ko nai dekhy ga.


Every array Method Syntax:

      array.every(function(currentValue, index, arr), thisValue)
      
Parameters:

      function()
                Required.
                A function to be run for each element in the array. 
      currentValue
                Required.
                The value of the current element.
      index
                Optional.
                The index of the current element.
      arr
                Optional.
                The array of the current element.
      thisValue
                Optional. Default undefined.
                A value passed to the function as its this value.

Some Examples:

Another Example with multiple values

      Example: Below is an example.

      const evenNumbers = [2,4,6,8,10];

      const result = evenNumbers.every((num) => {
        return num < 8; // yan par yeh "0,2 aur 4" ky lye True aur "6 aur 8" ky lye False ho ga.
      });
      console.log(result); // The Result is "False"
