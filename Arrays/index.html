Arrays in JavaScript

  • Arrays

    • With the help of an array, we can store more than one element under a single name.

    Syntax Of Arrays

          const arrayName = ["Value1", "Value", "Value", .....]

    Array Methods in JavaScript

    1. pop()

      2. Syntax: Array.pop()

      Is Method sy Hum End Wali Value ko Remove Kar Sakty Hain.

        const courses = ["Html", "Css", "JavaScript", "Python", "Java", "C++"]
  console.log(courses); // The Result is "(6) ['Html', 'Css', 'JavaScript', 'Python', 'Java', 'C++']
  courses.pop() // Remove End "C++" Value
  courses.pop() // Remove End "Java" Value
  console.log(courses); // The Result is "(4) ['Html', 'Css', 'JavaScript', 'Python']"

  Note! Is Method ko Jitni Dafa lagayn gy utni Value Remove Hoti Jayen gi.
    2. push()

      3. Syntax: Array.push(item1, item2, ..., itemX)

      Is Method sy Hum Array main New Value add kar Sakty Hain.

        const courses = ["Html", "Css", "JavaScript"]
  // Before Add Value Result
  console.log(courses); // The Result is "(3) ['Html', 'Css', 'JavaScript']"
  courses.push("Python", "Java")
  // After Add Value Result
  console.log(courses); // The Result is "(5) ['Html', 'Css', 'JavaScript', 'Python', 'Java']"

  Note! Is Main Hum aik Value b add Kar Sakty Hain or aik sy Ziaada b add Kar Sakty Hain.
    3. shift()

      4. Syntax: Array.shift()

      Is Method sy Hum Start Wali Value ko Remove Kar Sakty Hain.

        const courses = ["Html", "Css", "JavaScript", "Python", "Java", "C++"]
  console.log(courses); // The Result is "(6) ['Html', 'Css', 'JavaScript', 'Python', 'Java', 'C++']
  courses.shift() // Remove Start "Html" Value
  courses.shift() // Remove Start "Css" Value
  console.log(courses); // The Result is "(4) ['JavaScript', 'Python', 'Java', 'C++']"

  Note! Is Method ko Jitni Dafa lagayn gy utni Value Remove Hoti Jayen gi.
    4. unshift()

      5. Syntax: Array.unshift(item1, item2, ..., itemX)

      Is Method sy Hum Start Wali Value ko Return Kar Sakty hain jo Remove Hoi Thi. 

        const courses = ["Html", "Css", "JavaScript", "Python", "Java", "C++"]
  console.log(courses); // The Result is "(6) ['Html', 'Css', 'JavaScript', 'Python', 'Java', 'C++']
  courses.unshift() // Return Value Jo Remove Hoi thi mean "Css"
  courses.unshift() // Return Value Jo Remove Hoi thi mean "Html"
  console.log(courses); // The Result is "(4) ['JavaScript', 'Python', 'Java', 'C++']"

      Jitni Value Remove Hoi thi un ko Return Karny Ky Baad. Is Method ( unshift() ) ko Phir Lagayen gy tu
      Ik New Value Add Ho Jye Gi. 

        const courses = ["Html", "Css", "JavaScript", "Python", "Java", "C++"]
  console.log(courses); // The Result is "(6) ['Html', 'Css', 'JavaScript', 'Python', 'Java', 'C++']"
  courses.unshift("React") // New Value Add "React"
  console.log(courses); // The Result is "(7) ['React', 'Html', 'Css', 'JavaScript', 'Python', 'Java', 'C++']"

  Note! Remove Value ko Return Karny Ky Baad Jitni Dafa Is Method "shift()" ko Lagayn Gy New Value Add Hoti Jye Gi.
    5. concat()

      6. Syntax: Array1.concat(array2, array3, ..., arrayX)

      In This We Can Combine Different Arrays Into a Single Array

        const fruitsItems = ["Apple", "Banana", "Peach", "Dates"]
  const vegetablesItems = ["Ginger", "Garlic", "Potato", "Tomato"]
  const mixArray = fruitsItems.concat(vegetablesItems)
  console.log(mixArray);
  // The Result is "(8) ['Apple', 'Banana', 'Peach', 'Dates', 'Ginger', 'Garlic', 'Potato', 'Tomato']

  Note! Is Main Hum Different Arrays Ko Ik hi Array Main Jamam Kar Sakty Hain.
    6. indexOf()

      7. Syntax: Array.indexOf(item, start)

      In this we Find the Index of Values.

        const fruitsItems = ["Apple", "Banana", "Peach", "Dates", "Mango"]
  const indexOf = fruitsItems.indexOf("Dates")
  console.log(fruitsItems);
  console.log(indexOf); // This Fruit Item "Dates" IndexOf "3"

  Note! Is Main Hum Array ki Kisi b Value Ka Index Pata Karty Hain.
    7. includes()

      8. Syntax: Array.includes(element, start)

      In this we can Check if this Value is Included in the Array or Not. 

        const fruitsItems = ["Apple", "Banana", "Peach", "Dates", "Mango"]
  const includesTrue = fruitsItems.includes("Peach")
  console.log(includesTrue); // The Result is "True"
  const includesFalse = fruitsItems.includes("Lemon")
  console.log(includesFalse); // The Result is "False"

  Note! Is Main Hum yeh Pata Laga Sakty hain Ky Yeh Value Array Main Include Hai yan Nai.
  Note! Is Condition Main Hum yeh Pata laga Sakty hain Ky is Array main jo b hum search karyn wo include hai yan nai. 
  or is ka result "True" usi zoorat main mily ga jesa us main store kia ho ga. or ager kuch b galat type
  kia tu result "False" aye ga.
    8. reverse()

      9. Syntax: Array.reverse()

      Reverse Values After Make a Array.

        const fruitsItems = ["Apple", "Banana", "Peach", "Dates", "Mango"]
  const reverse = fruitsItems.reverse()
  console.log(fruitsItems);
  console.log(reverse); // The Result is "(5) ['Mango', 'Dates', 'Peach', 'Banana', 'Apple']"

  Note! Is Main Hum Array ki Values ko Reverse Kar Sakty Hain.
    9. sort()

      10. Syntax: Array.sort(compareFunction)

      In This We Can Sort the Values Alphabetically.

        const alphabets = ["G", "K", "I", "C", "J", "A", "E", "B", "F", "H", "D"]
  const sortWords = alphabets.sort()
  console.log(alphabets);
  console.log(sortWords); // The Result is "(11) ['A', 'B', 'C', 'D', 'E', 'F', 'G', 'H', 'I', 'J', 'K']" 
  

  Note! Yeh Original Array Ko b Change Kar Dy Ga.
    10. slice()

      11. Syntax: Array.slice(start, end)

      If we have to Select Some Values from an Array, We Can Separate the Values. 

        const animalsName = ["Dog", "Cat", "Cow", "Goat", "Camel", "Sheep", "Lion", "Tiger"]
  console.log(animalsName);
  const sliceValue = animalsName.slice(2, 6)
  console.log(sliceValue); // The Result is "(4) ['Cow', 'Goat', 'Camel', 'Sheep']"

  Note! Is sy Hum yeh Kar Sakty hain Ky Ager Hamy Array Main Sy Kuch Values Select Karni Hon Tu Hum Values ko 
          Separate Kar Sakty Hain.
    11. splice()

      12. Syntax: Array.splice( index, remove_count, .....item_list )

      If we have to Select Some Values from an Array, We Can Separate the Values.
      But it also Changes the Original Array, Meaning it Excludes Separate Values.

      JavaScript Array splice() Method is an inbuilt method in JavaScript that is used to change the contents of an array by removing or replacing existing elements and/or adding new elements. It modifies the original array and returns an array of the removed elements. 

        const animalsName = ["Dog", "Cat", "Lion", "Tiger", "Cow", "Goat", "Camel", "Sheep"]
  console.log(animalsName); // The Result is "(8) ['Dog', 'Cat', 'Lion', 'Tiger', 'Cow', 'Goat', 'Camel', 'Sheep']"
  const spliceValue = animalsName.splice(4)
  console.log(spliceValue); // The Result is "(4) ['Cow', 'Goat', 'Camel', 'Sheep']"
  console.log(animalsName); // The Result is "(4) ['Dog', 'Cat', 'Lion', 'Tiger']"

  Note! Is Main Hum Different Arrays Ko Ik hi Array Main Jamam Kar Sakty Hain.

      Remove Value for ( splice() Method help )

        const fruitItems = ["Apple", "Banana", "Kiwi", "Mango", "Grape"];
  console.log(fruitItems);
  const deleteItem = fruitItems.splice(2, 1) 
  console.log(deleteItem); // The "Kiwi" Item Removed
  console.log(fruitItems); // The Result is "(4) ['Apple', 'Banana', 'Mango', 'Grape']"

      Replace Value for ( splice() Method help )

        const fruitItems = ["Banana", "Kiwi", "Mango", "Grape"];
  console.log(fruitItems);
  const replaceItem = fruitItems.splice(1, 1, "Papaya") 
  console.log(replaceItem); // Replaced Value "Kiwi" to "Papaya"
  console.log(fruitItems); // The Result is "(4) ['Banana', 'Papaya', 'Mango', 'Grape']"

      New Add Multiple Values for ( splice() Method help )

        const fruitItems = ["Apple", "Banana", "Grape"];
  console.log(fruitItems);
  const addItem = fruitItems.splice(2, 0, "Papaya", "Water Melon") 
  console.log(addItem); // Added Two New Values "Papaya" and "Water Melon"
  console.log(fruitItems); // The Result is "(5) ['Apple', 'Banana', 'Papaya', 'Water Melon', 'Grape']"

    Multidimensional Arrays in JavaScript

    Multidimensional Arrays are also Called as Arrays inside Array.

    Syntax Of Multidimensional Arrays

       const arrayName = [[Array1], [Array2], [Array3], .....]

    Some Examples of Multidimensional Arrays

    Users Salary Example

          Example: Below is an example.
      
      const salary = [
        ["Ali", 500000],
        ["Bilal", 200000],
        ["Abdul Rehman", 100000]
      ];
      console.log(salary); // The Result is "(3) [Array(2), Array(2), Array(2)]"

    Kisi Array yan us ki Value ko is Tarah Access karayn gy.

          const salary = [
        ["Ali", 500000],
        ["Bilal", 200000],
        ["Abdul Rehman", 100000]
      ];
      const selectArray = [salary[1][0]] // The Result is "Bilal"
      console.log(selectArray);

      Note! Ager hum ik dafa asy "[1]" check karyn gy tu array ka index show ho ga or ager 2sri dafa type karyn gy tu array ky 
          ander value ka index show ho ga.
