While Loop

While Loop Syntax:

        while (condition) {
            Code block to be executed 
        }

Some Examples of While Loop

Counting Digits (1 to 5) Example

        Example: Below is an example

        let i = 1
        while(i <= 5){
            console.log(i);
            i++
        }

        Output in Console:
        1
        2
        3
        4
        5

Friends List Example

        Example: Below is an example

        const friendsName = ["Ali Hassan", "Bilal Hassan", "Qasim Hussain", "Usman Ali", "Saad Shabeer"]

        let i = 0;
        while (i < friendsName.length) {
            console.log(`${i + 1}. ${friendsName[i]}`);
            i++
        }

        Output in Console:
        1. Ali Hassan
        2. Bilal Hassan
        3. Qasim Hussain
        4. Usman Ali
        5. Saad Shabeer

For Loop

A for loop is used when you know how many times you need to repeat a certain block of code. It takes three statements.

For loop us time use hota hai jab ap janty hoon ky ap ko code ky ik makhsoos Block ko kitni bar run karna hai.

  1. Initialization statement

    2. It is the initialization of the counter. It is executed once before the execution of the code block.

  2. Condition Statement

    3. It defines the testing condition for executing the code block

  3. Increment or Decrement statement

    4. It is the increment or decrement of the counter & executed (every time) after the code block has been executed.

For Loop Syntax:

        for (statement 1 ; statement 2 ; statement 3){
            code here...
        }

Number Print (1 to 10) Example

        for(let i = 1; i <= 10; i++){
            console.log(i);
        }

Even Number (0 to 100) Example

        for(let i = 1; i <= 50; i++){
            console.log(`${i * 2}`) // The Result is "2,4,6,8,10,12,14,16,..........100"
        }

Counting Digits (1 to 10) Example

        for(let i = 1; i <= 10; i++){
            console.log(i);
        }Kk

Math Table Prints Example

        for (let i = 1; i <= 20; i++) {
            console.log('');
            console.log(`Table Number ${i}`);
            for (let j = 1; j <= 10; j++) {
                console.log(`${i} * ${j} = ${i*j}`);        
            }
        }

For of Loop

The for...of statement executes a loop that operates on a sequence of values sourced from an iterable object.

Note! For... of loop ka use hum Objects par nai kar sakty.

Note! Objects par loop karny kay lye hum For...in loop ka use karyn gy.

For...of loop Syntax:

        for (variable of iterable) {
            // code block to be executed
        }

        variable - For every iteration the value of the next property is assigned to the variable.
                    Variable can be declared with const, let, or var.

        iterable - An object that has iterable properties.

Looping over an Array

    	Example: Below is an example.

	const fruits = ["Apple", "Banana", "Peach", "Mango", "Water Melon"];
        for(const fruit of fruits){
        console.log(fruit)
        }
        
        Output:
        
        Apple
        Banana
        Peach
        Mango
        Water Melon

Looping over a String

    	Example: Below is an example.

	const sayHi = "Hello";
        for(const letter of sayHi){
        console.log(letter)
        }
        
        Output:
        
         H 
         e 
         l 
         l 
         o

For in Loop

For...in loop Syntax:

        for (key in object) {
            // code block to be executed
          }

UserIntro Example

        const userIntro = {
            firstName: "Abdul",
            lastName: "Rehman",
            age: 24,
            city: "Gujrat",
            skill: "Web Developer",
        }
        for(const key in userIntro){
            console.log(key)
        }

If only the keys of the object is to be accessed.

ager object ki sirf keys access karni hoon. 

    	Example: Below is an example.

        const userIntroKeys = Object.keys(userIntro)
        console.log(userIntroKeys)

        Output:

        (5) ['firstName', 'lastName', 'age', 'city', 'skill']

If only the values of the object is to be accessed.

ager object ki sirf values access karni hoon.

        const userIntroValues = Object.values(userIntro)
        console.log(userIntroValues)

        Output:

        (5) ['Abdul', 'Rehman', 24, 'Gujrat', 'Web Developer']

If both the keys and values ​​of the object are to be accessed.

ager object ki keys aur values dono access karni hoon. 

    	Example: Below is an example.

        const userIntroEntires = Object.entires(userIntro)
        console.log(userIntroEntires)

        Output:

         (5) [Array(2), Array(2), Array(2), Array(2), Array(2)]
           (2) ['firstName', 'Abdul']
           (2) ['lastName', 'Rehman']
           (2) ['age', 24]
           (2) ['city', 'Gujrat']
           (2) ['skill', 'Web Developer']

ForEach() Array Method

ForEach() loop Syntax:

        array.forEach(function(currentValue, index, arr), thisValue)
        
Parameters:

        function()
                  Required.
                  A function to run for each array element.
        currentValue
                  Required.
                  The value of the current element.
        index
                  Optional.
                  The index of the current element.
        arr
                  Optional.
                  The array of the current element.
        thisValue
                  Optional. Default undefined.
                  A value passed to the function as its this value.

Books Example with anonymous function and arrow function

        Example: Below is an example.

        const books = ["Urdu", "English", "Islamic Studies", "Physics", "Chemistry"];

        books.forEach((book) => {
          console.log(book);
        })

Animals Example with named function

        Example: Below is an example.

        const animals = ["Horse", "Cat", "Dog", "Tiger", "Lion"];

        function abc(el){
          console.log(el)
        }
        animals.forEach(abc) 

        Note! Named function itna common nai hai jitna anonymous function common hai.

Another ForEach Example with return value

        Example: Below is an example.

        const vegetables = ["Potato", "Cucumber", "Beetroot", "Garlic", "Ginger"];
        const sample = vegetables.forEach((vegetable) => {
          console.log(vegetable);
          return vegetable;
        });

        console.log(sample)

        Output:

        Potato
        Cucumber
        Beetroot
        Garlic
        Ginger
        undefined // This is Return value output
