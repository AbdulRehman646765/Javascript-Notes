Ternary Operators in javaScript

Conditional (Ternary) Operators

Syntax Of Ternary Operators

        const variableName = condition ? exprIfTrue : exprIfFalse;

Some Examples

        const username = 5 > 4 ? "Abdul Rehman" : "Bilal Hassan";
        console.log(username);
        

    
True Condition

        const gender = "M"
        const userMessage = `${gender.toLocaleLowerCase() === "m" ? "He" : "She"} Study in a College.`
        console.log(userMessage); // The Result is "He Study in a College." This True Condition

        Note! is main yeh hota hai ky jo b variable main save karyn gay. ager usy type karyn gy tu result "True" aye ga or 
           is ky ilawa kuch b type karyn gy tu result "False" aye ga.
        Note! jesa ky is example main hum ny "M" save kia hai or hum ny "M" type kia isi lye is ka result "True" mean "He" aya.
           or ager is ky ilawa kuch b type karty tu result "False" mean "She" ata.
            



False Condition

        const gender = "M"
        const userMessage = `${gender === "a" ? "He" : "She"} Study in a College.`
        console.log(userMessage); // The Result is "She Study in a College." This False Condition
