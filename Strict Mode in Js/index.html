Strict Mode in JavaScript

Strict Mode in JavaScript

Note! Strict Mode sy hamy Syntax errors ka pata chal jata hai. Ager hum ny Code main koi Mistakes kar di tu yeh Hamy bata hai ky ap ny kahan par Mistake ki hai.

Syntax Of Strict Mode: ( "use strict" )

Some Examples Of Strict Mode:

        a = 10;
        console.log(a); // The result is "10"

        function test(){
            "use strict";   
            b = 20;
            console.log(b); // The result is "b is not defined at test"
        }
        test()

        function test1(){
            "use strict";   
            let c = 30;
            console.log(c); // The result is "30"
        }
        test1()

        Note! In examples sy yeh baat Clear ho gai. Hum ny "a" sy pehly Strict Mode use ni kia tu Error ni mila. But hum ny 
        Function ky ander "b" sy pehly Strict Mode use kia tu Error mila.

        Note! Yeh Error is lye mily kio ky hum ny Variables ko "Declare" ni kia.
