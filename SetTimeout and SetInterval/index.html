SetTimeout and SetInterval in JavaScript

SetTimeout schedules a single execution after a specified delay, while SetInterval repeatedly executes a function with a fixed interval. SetTimeout is used for one-time delays or scheduling a task in the future, while SetInterval is suitable for recurring tasks.

SetTimeout or SetInterval dono same hain but in main yeh difference hai ky SetTimeout sirf aik dafa run hota hai. Or SetInterval bar bar run hota rehta hai.

SetTimeout

Definition:

SetTimeout schedules a single execution after a specified delay. SetTimeout is used for one-time delays or scheduling a task in the future

Note! SetTimeout ik esa Function hai jis ky ander koi String Pass karyn tu JavaScript usy Code main Change kar deta hai. Or Behind the Scene ik file Create karta hai. Or is ky ander pura Code jo hum ny String ky ander likha hai tu wo String ko Remove kar ky Normal Code kar deta hai.

Syntax: setTimeout(function, milliseconds)

Parameter:

  • function: the first parameter is a function to be executed
  • milliseconds: which indicates the number of milliseconds before the execution takes place.

Notes:

Note! SetTimeout main jab hum console.log main code likhyn gy tu console.log or jo is main code likhna hai is main yeh faraq karna hai mean yan Console.log ko Double Quotes ( "" ) yan is ky ander Code ko Double Quotes ( "" ) main likhna hai. Ager dono ko Double Quotes ( "" ) main likhyn gay tu Error aye ga. Ik ko Double Quotes ( "" ) main or ik ko Single Quote ( '' ) main likhna ho ga.

Example: setTimeout("console.log('Hi! My Name is Abdul Rehman')");

Note! Ager hum ny apny Code main ik hi line main ik sy ziaada console.log likhny hoon tu un ko Separate karny ky lye Semicolon (;) lagana ho ga nai tu Error aye ga.

Example: setTimeout("console.log('Hi!'); console.log('I am Abdul Rehman')");

Note! Ager Multiple line likhni hoon tu hum Backtick ( `` ) ka use karyn gay or kuch is tarah likhyn gay. Is main Semicolon (;) Zarori nai.

Example:
setTimeout(`
console.log('Hi!')
console.log('I am Abdul Rehman.')
console.log('I am 23 years old.')`);

Note! Ager hum pury Code ko Console ky ander Run karyn tu SetTimeout ki Return Value b ati hai. Yeh jo Return Value hai SetTimeout main jo Delay Time lagaya hai us ki Id hai.

Note! Ager aik sy ziaad SetTimeout lagayn tu sab sy End waly ki Id Show ho gi.

Note! Hum SetTimeout ki Id ko aik Variable main Store b kar sakty hain.
Note! Ager hum us Variable ko Run karyn gy tu Timer ki Id Show ho gi.

ClearTimeout:

Note! Ager hum ny kisi SetTimeout ko Remove karna ho tu hum ClearTimeout ki Help sy Remove kar sakty hain.

Example:
const timer1 = setTimeout(`console.log('Hi!-1')`, 1000)
const timer2 = setTimeout(`console.log('Hi!-2')`, 2000) // Timer2 Run nai ho ga kio ky hum ny isy Remove kar dya
const timer3 = setTimeout(`console.log('Hi!-3')`, 3000)

clearTimeout(timer2)

Output:
Hi!-1
Hi!-3

Note! Hum SetTimeout main Function b pass kar Sakty hain.
Note! Or jis main Function Pass Karyn gy wo aik Higher Order Function ban jye ga.

Example:
const timer1 = setTimeout(`console.log('Hi!-1')`, 1000)
const timer2 = setTimeout(`console.log('Hi!-2')`, 2000)
const timer3 = setTimeout(test, 3000) // This setTimeout is Higher Order Function

function test(){
console.log("Salam");
}

Output:
Hi!-1
Hi!-2
Salam

SetInterval

Definition:

SetInterval repeatedly executes a function with a fixed interval. SetInterval is suitable for recurring tasks.

Example:
const timer1 = setInterval(`console.log('Hi!-1')`, 1000) // This code will run after every Second
const timer2 = setInterval(`console.log('Hi!-2')`, 2000) // This code will run after every two second
const timer3 = setInterval(test, 3000) // This code will run after every three second
// This Timer3 is Higher Order Function

function test(){
console.log("Salam");
}

Output:
Hi!-1
Hi!-2
Salam

ClearInterval:

Note! Ager hum ny kisi SetInterval ko Remove karna ho tu hum ClearInterval ki Help sy Remove kar sakty hain.

Example:
const timer1 = setInterval(`console.log('Hi!-1')`, 1000) // Timer1 Run nai ho ga kio ky hum ny isy Remove kar dya
const timer2 = setInterval(`console.log('Hi!-2')`, 2000) // Run after every two second
const timer3 = setInterval(`console.log('Hi!-3')`, 3000) // Run after every three second

clearInterval(timer1)

Output:
Hi!-2
Hi!-3
