String Methods And Properties

Before starting string methods and properties we need know that strings are indexed. So basically we can access any single character of a string by doing something like this.

Property

There is only one property that exists.

  • length

Length Property

Is Property sy Hum Kisi b Value Ki Length Check Kar Sakty Hain.

        Example: Below is examples.
        
        const username = "Abdul Rehman";

Methods (Without Argument)

  • toUpperCase()

    • Is Method sy Hum Kisi b Value Ko Uppercase Kar Sakty Hain.

            Example: Below is an example.

          const username = "Abdul Rehman";
          const nameUpperCase = username.toUpperCase()
          console.log(nameUpperCase); // The Result is "ABDUL REHMAN"
  • toLowerCase()

    • Is Method sy Hum Kisi b Value Ko Lowercase Kar Sakty Hain.

            Example: Below is an example.

          const username = "Abdul Rehman";
          const nameLowerCase = username.toLowerCase()
          console.log(nameLowerCase); // The Result is "abdul rehman"
  • trim()

    • Is Method sy Hum Kisi b Value Sy White Space Ko Remove Kar Sakty Hain.

            Example: Below is an example.

          const username = "   Abdul Rehman   ";
          const nameTrim = username.trim()
          console.log(nameTrim); // The Result is "   Abdul Rehman   " to "Abdul Rehman"
  • trimStart()

    • Is Method sy Hum Kisi b Value Ky Start Sy White Space Ko Remove Kar Sakty Hain.

            Example: Below is an example.

          const username = "   Abdul Rehman";
          const nameTrimStart = username.trimStart()
          console.log(nameTrimStart); // The Result is "   Abdul Rehman" to "Abdul Rehman"
  • trimEnd()

    • Is Method sy Hum Kisi b Value Ky End Sy White Space Ko Remove Kar Sakty Hain.

            Example: Below is an example.

          const username = "Abdul Rehman   ";
          const nameTrimEnd = username.trimEnd()
          console.log(nameTrimEnd); // The Result is "Abdul Rehman    " to "Abdul Rehman"

Methods (With Argument)

  • includes()

    • In this we can Check if this Value is Included in the String or Not.

            Example: Below is an example.

          const username = "My Name is Abdul Rehman.";
          const includeNameTrue = username.includes("Abdul Rehman") 
          console.log(includeNameTrue); // The Result is True
          const includeNameFalse = username.includes("Abdur Rehman") 
          console.log(includeNameFalse); // The Result is False

      Note! is condition main hum yeh pata laga sakty hain ky is string main jo b hum search karyn wo include hai yan nai. 
          or is ka result "True" usi zoorat main mily ga jesa us main store kia ho ga. or ager kuch b galat type
           kia tu result "False" aye ga.
  • indexOf()

    • In this we Find the Index of Values.

            Example: Below is an example.

          const username = "Hello"; // is main 5 char hain or is ko count is tarah karyn gy (0,1,2,3,4)
          const indexOfName = username.indexOf("e"); 
          console.log(indexOfName); // The Result is "1"
      Note! wesy tu "e" 2sry number par hai or javaScript main choonky "0" sy count kia jata hai is lye yeh "1" number par hai.
  • replace()

    • In this we can Replace one Value with another Value.

            Example: Below is an example.

          const username = "0341 1264622"
          const replaceName = username.replace("0341 1264622", "0340 3148438")
          console.log(replaceName); // The Result is "0340 3148438"

      Note! is ka result tu yeh ana chahye tha "0341 1264622" par is ka result yeh "0340 3148438" is lye aya kio ky 
            hum ny "relpace()" method use kia.
  • replaceAll()
    •         Example: Below is an example.

          let text = "I love Dogs. Dogs are very easy to love. Dogs is a Beautiful Animal."
          text = text.replaceAll("Dogs", "Cats")
          console.log(text); // The Result is "I love Cats. Cats are very easy to love. Cats is a Beautiful Animal."

      Note! is main hum ny "Dogs" ko "Cats" sy "replace" kar dya. is method "replaceAll()" ki help sy.
  • concat()

    • In This We Can Combine Different Strings Into a Single String.

            Example: Below is an example.

          const username = "My Name is "
          const firstName = "Abdul"
          const LastName = "Rehman"
          const fatherName = "Muhammad Mehdi"
          const mixString = username.concat(firstName, " " , LastName, "." , " My Father Name is ", fatherName, ".");
          console.log(mixString); // The Result is "My Name is Abdul Rehman. My Father Name is Muhammad Mehdi."

      Note! is main hum different strings ko ik hi string main ekhatha kar sakty hain.
  • padStart()
    •         Example: Below is an example.

          const accountNumber = 8643
          const hideAccount = "8643".padStart(16, "*")
          console.log(hideAccount); // The Result is "************8643"

      Note! is main hum apny account number ko Hide kar sakty hain. ager number ko Start sy hide karna ho tu is method sy "padStart()"
          ki help sy kar sakty hain.
  • padEnd()
    •         Example: Below is an example.

          const accountNumber = 8643
          const hideAccount = "8643".padEnd(16, "*")
          console.log(hideAccount); // The Result is "8643************"

      Note! is main hum apny account number ko Hide kar sakty hain. or ager number ko End sy hide karna ho tu is method sy "padEnd()"
          ki help sy kar sakty hain.
  • charAt()
    •       Note! yeh method "indexOf()" ka opposite hai
        
        Example: Below is examples.
        
          const text1 = "Hello";
          const defineCharText1 = text1.charAt(1)
          console.log(defineCharText1); // The Result is "e"

          const text2 = "Hello";

      Note! is ko hum Uper dye howy 2no way "[]" or "charAt()" sy Define kar sakty hain.
  • charCodeAt()
    •         Example: Below is an example.

          const text = "Hello"
          const defineCharCode = text.charCodeAt(0)
          console.log(defineCharCode); // The Result is "72"

      Note! is main hum kisi b Character ka Code define kar sakty hain. choonky har character apna ik code rakhta hai.
  • split()
    •         Example: Below is an example.

          const username = "My Name is Abdul Rehman";
          const splitName = username.split(" ");
          console.log(splitName); // The result is "['My', 'Name', 'is', 'Abdul', 'Rehman']" 

      Note! is main hum ik bari string ko is method "split()" ki help sy chooti strings main Divide karty hain.
