Logical Operators in JavaScript

Logical Operators

  1. && (And)

    2. is condition main 2nu variables (isSchoolStudent & isCollegeStudent) equal mean True hony zaroori hain nai tu result False aye ga.

    Some Examples

                const userAge = 18;
            const isSchoolStudent = (userAge >= 5) && (userAge <= 18); // True Value
            const isCollegeStudent = (userAge >= 18) && (userAge <= 25); // True Value
            const isStudent = isSchoolStudent && isCollegeStudent;
            console.log(isStudent);
  2. || (Or)

    3. is condition main 2nu variables (isSchoolStudent & isCollegeStudent) main sy ager 1 b True ho tu result True aye ga.

    Some Examples

                const userAge = 25;
            const isSchoolStudent = (userAge >= 5) && (userAge <= 18); // False Value
            const isCollegeStudent = (userAge >= 18) && (userAge <= 25); // True Value
            const isStudent = isSchoolStudent || isCollegeStudent;
            console.log(isStudent);
  3. ! (Not)

    4. yeh operator "!" Truthy value ko False or Falsy value ko True kar deta hai.

    Some Examples

                const userAge = 25;
            const isSchoolStudent = (userAge >= 5) && (userAge <= 18); // False Value
            console.log(!isSchoolStudent); // True Value 
            const isCollegeStudent = (userAge >= 18) && (userAge <= 25); // True Value
            console.log(!isCollegeStudent); // False Value
